# Development Docs

## Quick Start

### Install Vue Modules and Plugins

- `npm install -g @vue/cli`
- `npm install -g @vue/cli-service-global`

### Build and Run

- `cd menuguide`
- `npm install`
- `npm run serve`

## Additional Vue Modules and Plugins Used
